# Research plan: Target checkout bots vs Target Checkout Helper

## Core question

How do public policy, technical patterns, and community discussion around Target (and retail) checkout automation compare to this repo’s architecture and risk profile?

## Subtopics

1. **Policy / ToS** — Target terms on automation, agents, scraping, buying agents, browsers.
2. **Technical patterns** — MV3 extensions, DOM vs API polling, failure modes vs our split (SW + content + MAIN world).
3. **Public landscape** — Monitors, CAPTCHA narratives, bans (anecdotal), marketing noise.

## Output

- Per-subtopic `findings_*.md` (completed by subagents).
- This folder’s `report.md` (synthesis with citations).
