Use .crx File

If you used GitHub "Code → Download ZIP"

After extracting the repo ZIP, run this file from the top-level folder:

target-checkout-helper-installer.exe

Keep payload files together

The installer will use one of these payload locations:

target-checkout-helper/ or target-checkout-helper.zip or dist/target-checkout-helper.zip

If you move the installer .exe out of the repo folder, move target-checkout-helper.zip next to it.

Alternative: single Windows bundle in dist/

If needed, use:

target-checkout-helper-installer-bundle.zip

Extract it to any folder on your PC.

Bundle contents

The extracted folder includes:

target-checkout-helper-installer.exe + target-checkout-helper.zip + INSTALL.html

Run the installer

Double-click target-checkout-helper-installer.exe. It extracts the extension and opens Chrome to chrome://extensions.

Enable Developer mode and load unpacked

In Chrome Extensions, turn on Developer mode, click Load unpacked, and select the extracted target-checkout-helper folder.

Pin & configure

Pin Target Checkout Helper, open the popup, toggle ON, fill your details, and click Save Settings.

Get the extension files

Clone or download this repository. The extension lives in the target-checkout-helper/ folder.

Alternatively, extract the zip from dist/target-checkout-helper.zip.

Open Chrome Extensions page

Type this into the Chrome address bar and press Enter:

chrome://extensions

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the page. Three new buttons will appear.

Click "Load unpacked"

Click the Load unpacked button (top-left), then navigate to and select the target-checkout-helper folder:

target-checkout-helper/
Important: Select the target-checkout-helper folder itself, not a file inside it.

Pin & configure

Click the puzzle-piece Extensions icon in the Chrome toolbar, then click the pin icon next to Target Checkout Helper.

Click the extension icon to open the popup. Toggle ON, fill in your shipping & payment details, and click Save Settings.

You're all set! Visit any product page on target.com and the extension will automate the checkout flow. It will never click "Place Order" — that's always your call.

Run the macOS/Linux installer

From repo root:

chmod +x install.sh
./install.sh

No sudo is required. The script extracts the extension (if needed) and opens Chrome Extensions.

Follow the prompts

The script will print the exact folder path to select. Click Load unpacked and paste or navigate to that path.

Pin & configure

Same as the manual method — pin the extension, open the popup, enter your info, and click Save Settings.

Windows users: use the Windows .exe Installer tab above for the simplest setup. install.bat is a fallback option.
Note: Modern Chrome restricts .crx sideloading. This method works best in Chromium, Brave, or Edge. For standard Chrome, use the "Load Unpacked" method above.

Locate the .crx file

The pre-built package is at:

dist/target-checkout-helper.crx

Open Extensions page

Navigate to chrome://extensions and enable Developer mode.

Drag & drop the .crx file

Drag target-checkout-helper.crx from your file manager and drop it onto the Chrome Extensions page. Confirm the installation prompt.

Pin & configure

Pin the extension, open the popup, and set up your checkout info.
